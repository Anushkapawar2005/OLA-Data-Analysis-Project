# 🚖 OLA Data Analysis Dashboard

## 📌 Project Overview

The **OLA Data Analysis Dashboard** is an interactive Power BI project designed to analyze ride booking data and provide meaningful business insights. The dashboard helps visualize booking trends, revenue, cancellations, customer behavior, vehicle performance, and ratings.

This project demonstrates practical skills in **Power BI**, **Power Query**, **DAX**, and **Data Visualization**.

---

## 🎯 Objectives

* Analyze ride booking trends over time.
* Monitor booking status.
* Compare revenue across payment methods.
* Identify the top customers by booking value.
* Analyze ride cancellation reasons.
* Evaluate customer and driver ratings.
* Create an interactive business dashboard.

---

## 📊 Dashboard Features

### 📍 Overall

* Ride Volume Over Time
* Booking Status Breakdown

### 🚗 Vehicle Type

* Top 5 Vehicle Types by Ride Distance
* Average Customer Rating by Vehicle Type

### 💰 Revenue

* Revenue by Payment Method
* Top 5 Customers by Total Booking Value
* Ride Distance Distribution Per Day

### ❌ Cancellation

* Customer Cancellation Reasons
* Driver Cancellation Reasons

### ⭐ Ratings

* Driver Rating Distribution
* Customer vs Driver Ratings
